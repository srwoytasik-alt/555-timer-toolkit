# 555 Timer Design Toolkit

Interactive calculators for designing circuits with the NE555 timer.

## Features

- Monostable timer calculator
- Astable oscillator calculator
- PWM generator calculator
- Adjustable delay calculator
- Tone generator calculator
- Bistable flip-flop reference
- Engineering unit formatting
- E12 and E24 resistor suggestions
- Astable design solver

## Circuits Included

- Monostable (One-Shot)
- Astable Oscillator
- PWM Generator
- Adjustable Delay
- Tone Generator
- Bistable Flip-Flop
